更改itunes文件备份路径

原文:http://jingyan.baidu.com/article/219f4bf7d71a9ede452d384b.html

在itunes里面，可以更改媒体资料库的路径，但是不能更改备份文件夹的路径。每次同步之后就会自动备份，然后就会在C:\Users\%username%\AppData\Roaming\Apple Computer\MobileSync下面生成一个很大的备份文件。为了减少C盘的占用率，特制作本教程。

  • 工具/原料
    • windows操作系统、itunes软件
    • NTFS格式的C盘
  • 方法/步骤
  • 打开C:\Users\%username%\AppData\Roaming\AppleComputer\MobileSync这个文件夹(%username%是指你开机时登录所用的用户名)；
  • 并把MobileSync下的全部文件剪切到你想存放的路径，如:F:\iTunes\MobileSync下（剪切前先关闭itunes软件）；
  • 删除C:\Users\%username%\AppData\Roaming\Apple Computer\MobileSync这个文件夹；
  • 打开cmd窗口，快捷键为win+r,然后输入cmd,再点击右键以管理员身份运行。输入以下命令： 
    mklink /J "C:\Users\%username%\AppData\Roaming\Apple Computer\MobileSync" "F:\iTunes\MobileSync"
# F:\iTunes\MobileSync是自己想要存放大文件的目录路径。
# 如果出现“创建的连接”表示成功。如果没有重复上面两个步骤即可。
# 成功后可以看到在C:\Users\%username%\AppData\Roaming\Apple Computer 下面MobileSync 的文件夹，其左下有一个箭头。
# 如果没有重复检查上述操作。
  • 注意事项

本经验在win7 64位下测试成功。其他windows系统适用。

iOS9关闭自动更新

  • 进入设置-通用-储存空间与iCloud用量-管理储存空间，删除已下载的iOS更新包。
  • 安装描述文件
    1. 打开你设备上的safari，然后访问 https://oldcat.me/web/NOOTA9.mobileconfig
    2. 按照提示输入密码、安装描述文件，提示重启的时候请重启。
    3. 然后就安装好了，不过桌面会多出一个“反馈助手”(Feedback)，你把它挪到某个文件夹无视即可。
    4. 这样，你应该已经检测不到更新啦(比如今天发布的iOS9.2.1，iOS10也是)。
  • 想删除本方法并恢复OTA更新的话，请到设置->通用，最下方找到描述文件，然后删除描述文件：
    • tvOS beta - Apple Developer v1
    • tvOS 10 beta Software Profile
