tree

  • tree命令乱码表问题
    • 前面的树状结构字符乱码，可能是终端charset有误造成中文显示乱码，解决方式：
      • 如:SecureCRT: Global Options -> Default Session -> Edit Default Settings... -> Terminal -> Character encoding... UTF-8
      • 或者使用--charset参数，改用ASCII字符：tree --charset ascii
    • 中文文件名乱码
      • 乱码实际是tree对non-printable字符的转义引起，使用-N选项即可解决：
        • -N Print non-printable characters as is instead of as escaped octal numbers.
      • tree -N
      • alias tree='tree -N'
