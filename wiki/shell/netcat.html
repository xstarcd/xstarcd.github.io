netcat

  • 远程shell 
    rm -f /tmp/f; mkfifo /tmp/f
cat /tmp/f | /bin/bash -i 2>&1 | nc -l 127.0.0.1 1234 > /tmp/f
  • 启动tcp侦听测试 
    # CentOS下安装nc工具
yum install nc

# 启动一个tcp侦听，并将收到的内容存入指定的文件中
# 需配置路由器等，确保该端口可以被互联网上任意主机访问到
nc -k -l 9090 >> updata

# 测试向该tcp端口发起连接、并发送数据，完成后比较发送端、接收端数据差异。
# 如：执行echo "abc" | nc IPaddress 9090，则updata的内容中将追加"abc"。
  • windows版本

    • 持继侦听一个端口：nc -L -p 9090 >> updata
