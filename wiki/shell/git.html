git将同一项目推送到两个仓库

  • 添加多个仓库方式 
    # 添加仓库
git remote add github https://github.com/demo.git
# 推送到仓库，推送两次
git push -u origin
git push -u gitee
  • 添加仓库并一次提交 
    # 添加仓库
git remote set-url --add origin https://github.com/demo.git
# 查看远程仓库配置，或直接查看.git/config中的配置
git remote -v
# 同步推送所有仓库
git push -u origin master
