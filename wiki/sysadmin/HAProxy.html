    timeout client          # 客户端数据传输超时时间
timeout server          # 后端服务器数据传输超时时间
timeout http-request    # 返回http头超时时间，未设置，默认timeout client
timeout http-keep-alive # keepalive超时时间，未设置，默认timeout http-request
timeout connect         # 连接超时时间
timeout check           # 未设置使用测试间隔时间(inter)作为测试的timeout(connect+read),
                        # 如果设置了:
                        #   使用测试间隔时间(inter)和连接超时(timeout connect)中最小的作为连接超时时间，
                        #   timeout check的时间作为读取时间
